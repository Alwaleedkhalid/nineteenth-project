# Ninetenths Assignment
 Laravel 5.8

## Installation
open new terminal 

Use following commands line in your termenal to start App.

```bash
$ composer install
```

```bash
$ composer update           
```
## Add new .env file

Use following commands line in your termenal to migration .

```bash
$ php artisan migrate:fresh
```

```bash
$ php artisan migrate:refresh --seed
```
